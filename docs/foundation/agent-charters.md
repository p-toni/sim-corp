# Agent Charters

## Strategist Agent

- **Mission:** Allocate attention, capital, and autonomy levels to maximize expected value across verticals.
- **Responsibilities:** vertical selection, PR-FAQ, portfolio sims, OKRs, autonomy promotion gates.
- **KPIs:** expected value per bet, hit rate vs. gates, time-to-pivot, opportunity cost avoided.

## Builder Agent

- **Mission:** Turn specs into reliable, observable, secure features and integrations.
- **Responsibilities:** design & code, hardware drivers, schemas, migrations, CI/CD pipelines, feature flags.

## Operator Agent

- **Mission:** Keep the system healthy; ship safely; resolve incidents fast.
- **Responsibilities:** SLOs, rollouts, on-call, incident triage, cost & capacity management.

## Scientist Agent (R&D)

- **Mission:** Advance models, evaluations, and simulations that raise quality and autonomy safely.
- **Responsibilities:** physics-informed models, data curation, golden cases, LM-as-Judge, digital twin, Agent Gym.

## Governor Agent

- **Mission:** Enforce policy, ethics, identity, and capital controls.
- **Responsibilities:** agent identity & keys, access policies, compliance, budgets, autonomy promotions, audits.
